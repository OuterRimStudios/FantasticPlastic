Interactive grass is a practical and mobile friendly grass rendering solution in unity3d.

"Interactive Grass" shader is the core implement of this asset.
It handle the movement of grass and support a burn out effect.
"Grass.cs" script manages a single grass, calculate the collision force with character and offer burn out control interface.

Open demo scene and click play button, Hold left mouse button and move can rotate the camera. Hold W,S,A,D,Q,E key will move the camera.
Hold direction key can move the capsule (aka "Character" in hierarchy window), it can interactive with the grass when collide happen.
Click "Burn Out" button on screen then you will see grass burn out from character position. Click "Reset" button revert all grass.
The demo scene demonstrates all features, please refer to it as example usage.
